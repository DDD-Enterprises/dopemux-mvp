## ⌨️ All Hotkeys

### Native DOPE Layout
- `M` - Toggle PM/Implementation mode
- `?` - Show help
- `Q` - Quit

### Added Enhancements
- `Ctrl+Alt+Left/Right/Up/Down` - Resize panes
- `Ctrl+Shift+Left/Right` - Swap panes
- `Ctrl+Alt+Z` - Toggle zoom
- `Ctrl+B, z` - Native zoom
- `v` (copy mode) - Begin selection
- `y` (copy mode) - Yank and exit

### PM Mode
- `↑↓` - Navigate task tree
- `←→` - Expand/collapse
- `Enter` - Select task

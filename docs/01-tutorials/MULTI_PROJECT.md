# Multi-Project Workflows

Use Dopemux workspaces and git worktrees to keep projects isolated.

- Generate per-workspace configs with `scripts/render_workspace_configs.py`.
- Switch worktrees instead of mixing project state in one workspace.

See the [Worktree Switching Guide](../04-explanation/concepts/WORKTREE_SWITCHING_GUIDE.md).
